A WordPress plugin that integrates Google's Material Design icons with the Gutenberg editor.

## Description

LWN Icons is a powerful WordPress plugin designed to seamlessly integrate Google's Material Design icons into your WordPress site. With this plugin, you can easily add modern and sleek icons directly within the Gutenberg editor, enhancing the visual appeal of your content.

## Installation

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/lwn-icons` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
